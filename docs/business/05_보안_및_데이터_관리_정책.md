# 전뇌비서관 보안 및 데이터 관리 정책
*더불어민주당 당원 전용 AI 서비스의 개인정보보호 및 보안 정책*

---

## 📋 정책 개요

### 서비스 정체성
- **서비스명**: 전뇌비서관 (전자두뇌비서관)
- **법인명**: 사이버브레인
- **서비스 성격**: 더불어민주당 당원 전용 정치 콘텐츠 생성 서비스
- **데이터 처리**: 정치 활동 관련 메타데이터 및 개인 선호도 학습

### 보안 철학
- **최소 수집 원칙**: 서비스 제공에 필요한 최소한의 정보만 수집
- **투명성**: 수집·처리·활용 목적의 명확한 고지
- **사용자 통제권**: 개인정보 열람·수정·삭제권 완전 보장
- **정치적 중립성**: 개인 정치 성향 데이터의 타 정당 유출 절대 차단

### 개인화 데이터 보안
- **Firestore 보안 규칙**: 사용자별 데이터 접근 제한 (본인 문서만 읽기/쓰기)
- **Firebase Auth 연동**: 인증된 사용자만 API 호출 가능
- **서버 측 처리**: 개인화 로직은 Cloud Functions에서 실행, 클라이언트에 노출 안 함

### 로그 관리 정책
- 개인정보가 포함되지 않은 감사 로그만 기록
- IP 주소 마스킹 처리
- 중요 작업에 대한 추적 가능성 확보
- 🆕 **개인화 관련 로그**: 메타데이터 접근, 모델 업데이트, 추천 생성 등 익명화하여 기록
- *로그 보관 기간 및 범위는 추후 결정*

## 🔮 미래 확장 계획

### 개인화 고도화 방향
- **현재**: 사용자 프로필 기반 프롬프트 개인화
- **다음**: 사용 패턴 기반 스타일 자동 학습

### 보안 고도화 방향
- **Firestore 보안 규칙 정기 점검**: 접근 제어 누락 방지
- **API 호출 제한(Rate Limiting)**: 비정상 사용 차단
- **Cloud Functions 로그 모니터링**: 이상 패턴 탐지
