# 전뇌비서관 사업 구조 및 운영 방침
*더불어민주당 당원 전용 AI 서비스*

## 📋 사업 구조 개요

### 브랜드 및 법인 구조
- **법인명**: 사이버브레인 (사업자등록)
- **공식 서비스명**: 전자두뇌비서관 (계약서, 공문서)
- **펫네임**: 전뇌비서관 (마케팅, 일상 대화)
- **브랜드 콘셉트**: 사이버펑크 + 정치권 전용 (공각기동대 오마주)

### 사업자 형태
- **개인사업자 단독 운영**
- **투자 유치 없음**
- **지분 분할 없음**
- **모든 수익 단독 귀속**

## 🏢 운영 방침

### 1인 자동화 운영 원칙

#### 핵심 운영 철학
- **"혼자서도 운영 가능한 시스템 구축"**
- **"복잡한 조직 구조 지양"**
- **"자동화를 통한 효율성 극대화"**
- **"단순하고 명확한 사업 구조 유지"**

#### 자동화 영역
- **AI 원고 생성**: Gemini API 자동 처리
- **결제 관리**: 토스페이먼츠 자동 연동
- **당적 인증**: OCR 자동 처리
- **고객 관리**: Firebase 자동화 시스템
- **서버 운영**: Firebase 서버리스 아키텍처

#### 수동 관리 영역 (최소화)
- **고객 지원**: 챗봇 + 개인 연락처 (카톡/전화)
- **영업 활동**: 당내 네트워크 직접 활용
- **품질 관리**: 정기적 프롬프트 개선
- **정책 업데이트**: 서비스 정책 조정
- **브랜딩 관리**: 전뇌비서관 펫네임 확산 활동

## 💰 재무 운영 방침

### 수익 귀속 구조
```
전뇌비서관 매출 → 개인사업자 수입 → 개인 소득
                                    ↓
                              소득세 납부 후 순수익
```

### 투자 정책
- **❌ 외부 투자 유치 금지**
- **❌ 지분 분할 불허**
- **❌ 파트너십 지분 제공 불허**
- **✅ 협력은 용역/위탁 계약으로만**

### 재무 관리 원칙
- **개인사업자 단순 구조 유지**
- **법인 전환 계획 없음** (자유로운 수익 운용 우선)
- **복잡한 절세 구조 지양** (93% 마진에서 세금은 감수)
- **홈택스 직접 신고** (단순한 사업 구조로 셀프 관리 가능)
